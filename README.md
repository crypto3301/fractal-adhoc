# Симулятор Ad-Hoc сети с предфрактальной топологией

**Задание:** Разработать программу, объединяющую случайное количество сетевых узлов в Ad-Hoc сеть с топологией, соответствующей предфрактальному графу определенного порядка.

## Требования

- Визуальное представление Ad-Hoc сети с предфрактальной топологией
- Отображение графа топологии в нормированном виде
- На первом этапе узлы в случайном порядке выводятся на экран
- Далее узлы объединяются в сеть по правилам построения предфрактальных графов
- Возможность просмотра индекса каждого узла в рамках предфрактальной топологии
- Количество узлов: не менее 64
- Порядок графа: не менее 3
- Затравка: полносвязный четырехугольник (K4)
